# **App Name**: Loyalty Leap Dashboard

## Core Features:

- Total Points Liability KPI: Calculate and display the total points liability (sum of all customer's `totalPointsBalance`).
- Total Registered Customers KPI: Calculate and display the total number of registered customers.
- Top 10 Customers Table: Display a table showing the top 10 customers with the most points, joined with their corresponding membership level name.
- Sales Report Generation: Generate a downloadable .CSV sales report based on data from the `Purchases` collection.  Uses PapaParse tool.
- Firestore Integration: Integrate with Firestore database to read customer, purchase, and membership data.

## Style Guidelines:

- Primary color: Deep Blue (#3F51B5) to convey trust and stability, reflecting the reliability of the loyalty program.
- Background color: Light Gray (#F5F5F5), providing a clean and neutral backdrop.
- Accent color: Orange (#FF9800) for highlighting key actions and information, complementing the primary blue.
- Font pairing: 'Inter' (sans-serif) for both headlines and body text to provide a modern and readable experience.
- Responsive grid layout with clear separation of KPIs, tables, and reports, optimizing the display for various screen sizes.
- Use simple and professional icons from a library like FontAwesome or Material Icons to represent different data points and actions.
